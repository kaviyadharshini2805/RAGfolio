3. ResumeRAG
Description:
A Retrieval-Augmented Generation (RAG) application designed to answer questions about resume content. The system extracts text from PDF resumes using PyPDF2, chunks the content into manageable segments, generates vector embeddings using Sentence Transformers, stores them in FAISS vector database for efficient similarity search, and retrieves relevant context to answer user queries using a Large Language Model. Supports multiple resume uploads and maintains semantic understanding of professional experience, skills, and qualifications.

Role:
Solo Developer - Full implementation of RAG pipeline

Tech Stack:
Python, FAISS, Sentence Transformers, PyPDF2, LLMs

Key Achievements:
- Implemented PDF text extraction and preprocessing pipeline
- Built FAISS vector database for efficient semantic search
- Generated high-quality embeddings using Sentence Transformers
- Designed context retrieval system with relevance ranking
- Integrated LLM for natural language question answering
- Created modular architecture for easy maintenance and scaling

Skills Demonstrated:
RAG Architecture, Vector Databases, Document Processing, Semantic Search, LLM Integration, PDF Parsing, Information Retrieval

5. Elafina - Personal AI Chatbot
Description:
A production-ready personal portfolio chatbot built using Retrieval-Augmented Generation (RAG) architecture. The system ingests personal information from structured text files (resume, skills, projects, experience, education), chunks and embeds the content using HuggingFace Sentence Transformers (all-MiniLM-L6-v2), stores vectors in ChromaDB for persistent semantic search, and uses Google Gemini (gemini-2.5-flash) as the LLM for natural conversational responses. Built with modular architecture separating concerns: data loading, embedding generation, vector database management, RAG pipeline, and chatbot interface. Supports real-time knowledge base refresh and maintains conversation history.

Role:
Solo Developer - Designed and implemented complete RAG system from scratch

Tech Stack:
Python, Streamlit, ChromaDB, LangChain, HuggingFace Embeddings (Sentence Transformers), Google Gemini API, LangChain Expression Language (LCEL)

Key Achievements:
- Built complete RAG pipeline with modular architecture (loader, embeddings, vectordb, RAG, chatbot modules)
- Implemented semantic search using ChromaDB with persistent storage
- Integrated HuggingFace embeddings for free local embedding generation (no API costs)
- Used Google Gemini API for chat responses with cost-effective model selection
- Created interactive Streamlit UI with conversation history and knowledge base management
- Handled document chunking with overlap for better context retrieval
- Implemented proper error handling and environment configuration
- Achieved accurate context-aware responses about professional background

Skills Demonstrated:
RAG Architecture, Vector Databases, Semantic Search, LLM Integration, Prompt Engineering, Embedding Generation, Backend Design, API Integration, Chatbot Development, Full-Stack Development
